Inteligencia Artificial Reconocimiento de Rostro

En el primer Proyecto se quiso poner en práctica el funcionamiento de un API KEY en React el cual conecta con una inteligencia artificial la cual se encarga de señalar el rostro proporcionando el link de una imagen, también contando con un inicio de sesión en el cual se almacena la información en una base de Datos PostgresQL, la cual almacena los usuarios registrados, además de ir sumando un puntaje a medida que se van introduciendo imágenes parecido a un Score, para los detalles de cómo poner en marcha la app ingrese al código fuente y lea el README de GitHub.

Inicio de Sesión con google, y aplicación de Notas

Se realizó una aplicación en React, la cual se caracteriza de manejar un sistema de inicio de sesión real, el cual te da la opción de iniciar sesión con una cuenta google mediante un API. Luego del inicio de sesión hay una interfaz creada con SASS en cuanto a diseño, la cual te permite agregar notas, una descripción, adjuntar una imagen, guardar y eliminar las notas, al guardarla se queda registrada en la cuenta con la que se inicio sesion, en esta se aplican muchos conceptos importantes, como la implementación de Redux, el funcionamiento de los Hooks, Firebase y cuenta con su apartado de pruebas, detalles en el README de github.

Buscador de Productos por Id y por Nombre

En este proyecto se realizó una aplicación web la cual se encarga de poder buscar productos por el Id y por el nombre de este, contando además con filtros de búsqueda que prioriza los productos con descuento, contando con servicios Rest para el backend. Esta tiene una lista Json de productos la cual se unificó con Docker generando una base de Datos Mongodb, aproximadamente hay 2 mil productos en la base de datos, este programa está completamentamente configurado con Docker, lo cual hace mucho más fácil e intuitivo el manejo de este, se desarrolló en Angular y cuenta con pruebas unitarias de Angular, para poder correr esta app solo se necesita Docker, detalles en el README de GitHub.
